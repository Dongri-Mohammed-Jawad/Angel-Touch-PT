A) Overview - - top

Before we start, lets take a look on the folder structure of the theme. All theme files are placed in the HTML folder. The main HTML folder consists of assets folder for different types of theme assets:


  1. css - contains CSS files for the theme
  2. fonts - contains @fontface for the font customization
  3. img - contains images for the theme design
  4. js - contains javascript files for the theme functionality

B) Installation - top

This is an HTML theme and doesn't require instalation in the traditional meaning of the word. You only need to place the content (HTML files and the three folders) in your prefered web directory


Keep in mind that the folder structure and file/folder names are important for the correct theme work

D) CSS Files and Structure - top

There are three css files necessary for customization.

  1. "assets/css/core/bootstrap.min.css" -Is the bootstrap main css file.
  2. "Using CDN" -Is the Font awesome css file.
  3. "https://fonts.googleapis.com/css?family=Lato:300,400,700|Raleway:400,700" -Is the Google Fonts Import Link.
  4. "assets/css/external/animate.min.css" -Is the animate css file.
  5. "assets/css/external/jquery.bxslider.css" -Is the bxslider css file.
  6. "assets/css/external/flexslider.css" -Is the flexslider css file for sliders.
  7. "assets/css/external/slick.css" -Is the slick css file for sliders.
  8. "assets/css/external/morphext.css" -Is the morphext css file for Text Rotators.
  9. "assets/css/external/ytplayer.css" -Is the ytplayer css file for fullscreen background video.
  10. "assets/css/external/yamm.css" -Is the yamm css file for full-width dropdowns
  11. "assets/css/external/lightbox.css" -Is the lightbox css file.
  12. "assets/css/project/styles.css" -Is the main css file.
